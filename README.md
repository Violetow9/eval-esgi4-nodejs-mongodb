## Init

### Requirements
- Node.js version 21 or higher
- NPM

### Setup
1) Copy and paste the following code into your terminal...
    ```sh
    cp .env.example .env
    npm i
    ```
2) Fill the corresponding .env values in your .env file
3) Execute ``npm run start`` and the project is ready to go !
